# Chat Node.js

Chat desenvolvido no período da pandemia com intuito de implementar uma comunicação cliente/servidor
em que é possível realizar comunicação privada, em grupo e com todos usuários conectados.

## Desenvolvedores

- Nicolas Bortoluzzi
- Iago Almeida

## Tecnologias Usadas

- Node.js
- Vue.js
- Socket.io

## Como Usar

`git clone https://github.com/Nicolasbort/Chat-nodejs`</br>
`cd Chat-nodejs`

- Abra dois terminais

### Em um terminal

`cd client`</br>
`npm install`</br>
`npm run serve`

- Abra http://localhost:8080

### No outro temrminal

`cd server`</br>
`npm install`</br>
`npm run start`
